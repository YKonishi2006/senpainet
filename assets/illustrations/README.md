# イラスト素材の置き場所

ここに画像ファイルを置いてください。置いたら「置きました」と教えていただければ、
サイト各所に差し込みます（現在はすべて自作SVGの仮イラストです）。

---

## 方法A（おすすめ）：12点を個別ファイルで保存

貼っていただいた素材シートの各イラストを、**下の名前**で保存してください。
形式は **PNG（背景透過）推奨**、SVGならさらに良いです。

| ファイル名 | 素材（シート上の位置） | 使う場所 |
|---|---|---|
| `boy-laptop.png` | 1段目1列：PCの男子＋ハート | 先輩の方へ ヒーロー／「先輩が回答」 |
| `girl-book.png` | 1段目2列：本を持つ女子 | 在校生／「相談する」 |
| `chat.png` | 1段目3列：吹き出し2つ | 相談・やりとりのイメージ |
| `cap-books.png` | 1段目4列：卒業帽と本 | 進路・大学／卒業生 |
| `mail.png` | 2段目1列：封筒＋通知バッジ | お知らせ／通知 |
| `checklist.png` | 2段目2列：チェックリスト | 投稿前チェックリスト／安心・安全 |
| `growth.png` | 2段目3列：右肩上がりのグラフ | 実績／運営ダッシュボード |
| `trophy.png` | 2段目4列：トロフィー | 実績・貢献 |
| `medal.png` | 3段目1列：星のメダル | 評価・「役に立った」 |
| `highfive.png` | 3段目2列：ハイタッチ | 先輩と後輩がつながる／CTA |
| `paper-plane.png` | 3段目3列：紙飛行機 | 投稿・送信／投稿完了 |
| `girl-phone.png` | 3段目4列：スマホの女子＋ハート | ホーム／私たちについて ヒーロー |

---

## 方法B（最短）：シート1枚をそのまま保存

12点が **4列×3行** に均等配置されているので、シート画像を1枚だけ
`illustrations.png` という名前でここに保存していただければ、
こちらで **CSSスプライト**（背景位置で切り出し）として各所に使います。

※ 方法Aの方が高品質・扱いやすいです。方法Bは手軽ですが、切り出し位置の微調整が必要な場合があります。

---

## 補足
- 全部そろわなくてOKです。置いた分だけ差し替えます。
- 画像は GitHub Pages でそのまま配信されます（外部依存なし）。
- サイズの目安：1点あたり 600〜1000px 程度（大きすぎる場合はこちらで調整します）。
