# SenpaiNet

同じ学校の先輩（在校生・卒業生）に匿名で相談できる、中高一貫校向けの相談プラットフォーム（プロトタイプ）。

単一の `index.html` で動作する自己完結型のデモです。ブラウザで `index.html` を開くか、GitHub Pages で公開してご覧ください。

## 主な画面
- ホーム / 相談を探す（在校生→先輩） / 相談する
- 先輩の方へ / 卒業生コミュニティ / 先輩からの募集・お誘い
- お知らせ / 使い方 / サポート・利用規約・プライバシー / マイページ

在校生・卒業生の区分は「回生」から現在の年度で自動計算され、毎年度くり上がります。
